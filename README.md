# World Cup Predictor

A full-stack FIFA World Cup 2026 prediction platform built with:

## Frontend
- React
- TypeScript
- Vite
- Tailwind CSS

## Backend
- FastAPI
- SQLAlchemy
- JWT Authentication

## Features
- User Registration/Login
- Group Stage Predictions
- Knockout Bracket Predictions
- Admin Panel
- PDF Export
- Leaderboard

## Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
